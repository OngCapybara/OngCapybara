In this blog post, I will share some Windows shortcuts that will help make things easier for you without having to interact with the touchpad too often. I always use these shortcuts to enhance the aesthetics of my desktop display. By using these shortcuts, I hope to broaden your knowledge and increase your daily creativity. BTW, if you are good at using shortcuts, you will look like a pro lol XD.


1. Open/Close Software

Ctrl + Shift + Esc    # Open Task Manager
Win + 1/2/3/4    # Open Software On Taskbar
Win + D    # Hide
Win + E    # Open File Explorer
Win + I    # Open Settings
Win + R    # Open Run Command Window
Win + Shift + W    # Closed Software

2. Screen Option

F11    # Full Screen
Win + L    # Lock Screen
Win + M    # Minimize
Win + P    # Open Pc Screen Setting (If you used some monitors)
