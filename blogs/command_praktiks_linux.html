Ini kumpulan tips dan snippet Linux command yang sering dipakai ‚Äî dari operasi dasar (shutdown, reboot), membuka aplikasi, sampai beberapa trik productivity yang berguna untuk development dan administrasi. Semua contoh ditulis agar mudah dicopy-paste ke terminal atau dimasukkan ke `~/.bashrc` / `~/.zshrc`.

Dasar Cepat: Shutdown, Reboot, Suspend

Perintah-perintah berikut umum dipakai dan biasanya membutuhkan sudo (kecuali user memiliki privilege khusus).

# Matikan sekarang
sudo shutdown -h now

# Reboot cepat
sudo reboot
# atau dengan systemd
sudo systemctl reboot

# Suspend (sleep)
systemctl suspend

# Hibernate
sudo systemctl hibernate

# Batalkan scheduled shutdown
sudo shutdown -c

Buka Aplikasi / File dari Terminal

Buka file/folder atau jalankan GUI app dari terminal dengan cara yang tidak mengunci terminal.

# Buka file/folder dengan aplikasi default
xdg-open file.pdf
xdg-open .

# Jalankan GUI di background supaya terminal kembali
gnome-calculator &      # contoh
code . &                # VSCode
firefox 'https://google.com' &

# Jalankan dan biarkan jalan setelah logout
nohup some-gui-app > /dev/null 2>&1 &
disown

Navigasi & File Ops (Sering Dipakai)

# List human-readable
ls -lah

# Copy / move / rename
cp source.txt dest.txt
mv oldname newname

# Hapus dengan konfirmasi
rm -i file.txt

# Buat direktori beserta parent
mkdir -p path/to/folder

Proses & Port ‚Äî Cari & Kill

Tracing proses yang pakai port atau menghabisi proses yang membandel.

# Lihat proses yang pakai port 8080
sudo ss -tulpn | grep :8080
# atau
sudo lsof -i :8080

# Cari proses dan kill
ps aux | grep myserver
kill 
kill -9    # paksa (hati-hati)

# Monitor proses (lebih enak pakai htop)
top
htop

Cari Teks & File

# Cari teks di file (rekursif)
grep -Rin "needle" path/

# Cari file berdasarkan nama
find . -name "*.log"

# Cari file lalu tampilkan baris yang cocok
find . -name "*.conf" -exec grep -H "Listen" {} \;

Clipboard & Edit Cepat

# Copy ke clipboard (butuh xclip / xsel)
echo "hello" | xclip -selection clipboard
cat file.txt | xclip -selection clipboard

# Paste clipboard ke file
xclip -o -selection clipboard > file.txt

# Editor cepat
nano file.txt

Permission & Owner

# Ganti permission
chmod 755 script.sh
chmod u+x script.sh

# Ganti owner
sudo chown user:group file.txt

Productivity: History, Aliases & Shortcuts

Beberapa alias dan trik singkat yang hemat waktu ‚Äî tambahkan ke ~/.bashrc atau ~/.zshrc.

# Ulang perintah terakhir dengan sudo
sudo !!

# Jalankan perintah dari history
history
!123

# Contoh alias (taruh di ~/.bashrc atau ~/.zshrc)
alias ll='ls -lah'
alias gs='git status'
alias shutdownnow='sudo systemctl poweroff'
alias serve='python3 -m http.server 8000'

# Setelah edit, reload
source ~/.bashrc

Snippet Helper yang Berguna

# Buka VSCode di folder lalu background
(code . & disown)

# Simple HTTP server (share folder cepat)
python3 -m http.server 8000

# Restart service (systemd)
sudo systemctl restart nginx
sudo systemctl status nginx

Tips Keamanan & Kebiasaan Baik

  • Jangan sembarangan pakai sudo rm -rf ‚Äî cek path dulu.
  • Gunakan -i saat hapus file bila ragu: rm -i file.txt.
  • Untuk server production, hindari menjalankan GUI via SSH kecuali paham X forwarding / nohup.
  • Gunakan user DB dengan hak minimum pada production (least privilege).
